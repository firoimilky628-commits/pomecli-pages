PomeCLI Support

Last updated: 2026-06-21

English

PomeCLI is an iPhone / iPad terminal client for connecting to your own Mac or server over SSH and operating tmux sessions.

For support, bug reports, or App Store review questions, contact:

firoimilky628@gmail.com

Before contacting support

  • Remote Login / SSH is enabled on your Mac or server.
  • The host name or IP address, port, user name, and password are correct.
  • The server is reachable from your iPhone / iPad network.
  • tmux is installed on the server.
  • If a host key warning appears after rebuilding or replacing the server, verify that you are connecting to the intended server.

PomeCLI does not operate a hosted SSH service. You must provide and manage your own SSH server.

問い合わせ前の確認

  • Mac またはサーバーでリモートログイン / SSH が有効になっている。
  • ホスト名または IP アドレス、ポート、ユーザー名、パスワードが正しい。
  • iPhone / iPad のネットワークからサーバーへ到達できる。
  • サーバーに tmux がインストールされている。
  • サーバーの再構築や交換後にホスト鍵の警告が出た場合、意図したサーバーへ接続していることを確認する。

PomeCLI はホスト型の SSH サービスを提供しません。接続先の SSH サーバーはユーザー自身で用意・管理する必要があります。
